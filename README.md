# Dream Projects — Landing

Sitio estático (HTML/CSS/JS, sin build) del estudio de productos digitales **Dream Projects**.

## Estructura
- `index.html` — landing principal (hero, servicios, productos, proceso, contacto)
- `terms.html` / `privacy.html` — páginas legales (necesarias para activar pagos)
- `styles.css` — estilos y animaciones
- `app.js` — interacciones (nav, reveal on scroll, contadores, parallax)

## Deploy
Sitio estático. En Render: tipo **Static Site**, *Build Command* vacío, *Publish Directory* `.`.
También sirve para GitHub Pages o cualquier hosting estático.

## Personalizar
- Marca y textos: editar `index.html`.
- Email de contacto: buscar `sacebado@gmail.com`.
- Colores: variables CSS al inicio de `styles.css` (`--violet`, `--cyan`, `--pink`).
